Voor de eindopdracht van de module Basic IT Skills vragen we je om, voor de deelopdracht rond Computerarchitectuur, volgende vragen uit te werken:

* Er wordt al eens gezegd dat busbreedte belangrijk is, licht dit toe en geef een passend voorbeeld.

Zoals in de cursus uitgelegd, hoe groter de busbreedte hoe meer gegevens per keer getransporteerd kunnen worden, zoals het cursusvoorbeeld hoe meer rijvakken,hoemeer autos er gebruik van kunnen maken.

* Heeft Windows een non pre-emptive of pre-emptive CPU scheduler? Wat is het verschil?

preemptive omdat als je kijkt naar mijn volgende vraag kan een actie onderbroken worden met hogere prioriteit in een non preemptive systeem gaat dit niet

* Wat is een Interrupt Request? Licht toe aan de hand van een voorbeeld.

veel apparaten willen regelmatig gegevens versturen naar de processor,een apparaat kan deze gegevens niet zomaar op de databus zetten , want er kan op datzelfde moment al een ander apparaat aan het zenden zijn , hoe lager het irq nummer hoe hoger de prioriteit en deze hebben dan voorrang op interrupts met een lagere prioriteit.

* Wat is een register?

kleine interne geheugens die zeer snel gelezen of beschreven kunnen worden omdat ze zich in de cpu bevinden

* Leg uit: Waar staat de afkorting RAM voor? Waarvoor wordt RAM gebruikt?

Ram staat voor Random access memory, Ram word gebruikt voor tijdelijke bestanden op te slaan, hoe meer ram geheugen, des te meer processen je tegelijk uit kan voeren
